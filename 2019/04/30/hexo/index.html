名称 作用 操作
hexo-douban 关联豆瓣阅读、电影等 1. npm install hexo-douban --save；
2. 在root的_config.yml中写入配置项
3. 使用该插件若弹出空白页，将_config.yml文件中站外链接新窗口打开设为false
local_search
Algolia		 站内文章搜索 1. npm install hexo-generator-search --save；
2. 在root的_config.yml中写入配置项
LeanCloud
Valine 简约评论系统，依托于LeanCloud * 目前大部分Hexo主题提供支持
1. 获取LeanCloud中AppID和APPKey
2. 修改配置初始化内容为上值

mathjet3

2. 设置_.yml加载katex.min.css
hexo-footnotes 插入脚注、引用
